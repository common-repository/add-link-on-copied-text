== Description ==

This simple plugin adds a link to the text copied from your site. It starts working immediately after installation and activation, but it also has a few simple optional settings for customization.

== Installation ==

1. Upload dir `add-link-on-copied-text` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. View on the post page.

== Frequently Asked Questions ==

= What does this plugin do? =
The plugin automatically adds a link to your site at the end of the text copied by any visitor. You can configure several options for the link and its display in the plugin settings.

= Why do I need this plugin? =
When someone takes part of your content from the site, you want it to be not in vain, and this plugin is created for this.

When someone copies text to your site, you can remind the user that the text is your property and respecting your work, the person copying should at least paste a link to you.

Also, the text may appear in the correspondence between users and encourage them to visit your site, and some programs automatically collect texts from sites and having the “Add link on copied text” plugin installed, you can get more incoming links to your website.

= Is the plugin compatible with my site? =
Depends on your site, but this plugin is not very complex from a technical point of view, it was tested with the latest version of WordPerss and no errors were noticed.

= What plugins do you recommend installing along with "Add link on copied text"? =
I also have other plugins that will help make your site a little better and more convenient. You can get acquainted with them by finding my profile on WordPress.org by nickname @dmitrylitvinov. I am sure you will find there other technical solutions that you will need.

= How to install the plugin? =
Download the plugin from the admin panel and then activate it. Or download the plugin from the official WordPress site, upload it to your site and then activate it.
